# 一、创建第一个应用程序

## 1.1 版本需要

- JDK11+
- Maven3.6.2+



## 1.2 项目创建

### 1.2.1 下载GIT存储库

```shell
git clone https://github.com/quarkusio/quarkus-quickstarts.git
```



### 1.2.2 使用mvn创建

1. Linux & MacOS 用户

   ```shell
   mvn io.quarkus:quarkus-maven-plugin:999-SNAPSHOT:create \
       -DprojectGroupId=org.acme \
       -DprojectArtifactId=getting-started \
       -DclassName="org.acme.getting.started.GreetingResource" \
       -Dpath="/hello"
   cd getting-started
   ```

2. windows

   ```shell
   mvn io.quarkus:quarkus-maven-plugin:999-SNAPSHOT:create -DprojectGroupId=org.acme -DprojectArtifactId=getting-started -DclassName="org.acme.getting.started.GreetingResource" -Dpath="/hello"
   ```

   ```shell
   mvn io.quarkus:quarkus-maven-plugin:999-SNAPSHOT:create "-DprojectGroupId=org.acme" "-DprojectArtifactId=getting-started" "-DclassName=org.acme.getting.started.GreetingResource" "-Dpath=/hello"
   ```

3. 通过网站创建

   [点击打开](https://code.quarkus.io)

## 1.5 编写代码

```java
@ApplicationScoped
public class GreetingService {

    public String greeting(String name) {
        return "hello " + name;
    }
}




@Path("/hello-resteasy")
public class GreetingResource {
    @Inject
    GreetingService service;

    @GET
    @Produces(MediaType.APPLICATION_JSON)
    @Path("/greeting/{name}")
    public String greeting(@PathParam("name") String name) {
        return service.greeting(name);
    }

    @GET
    @Produces(MediaType.TEXT_PLAIN)
    public String hello() {
        return "Hello RESTEasy";
    }
}
```

```java
@QuarkusTest
public class GreetingResourceTest {

    @Test
    @DisplayName("测试hello接口")
    public void testHelloEndpoint() {
        given()
          .when().get("/hello-resteasy")
          .then()
             .statusCode(200)
             .body(is("Hello RESTEasy"));
    }

    @Test
    @DisplayName("测试新编辑接口")
    public void testGreetingEndpoint() {
        String uuid = UUID.randomUUID().toString();
        given()
                .pathParam("name", uuid)
                .when().get("/hello-resteasy/greeting/{name}")
                .then()
                .statusCode(200)
                .body(is("hello " + uuid))
        ;
    }
}
```
